# MEMEWARS ARENA BIBLE (CANON)


What it is:
MemeWars is the team’s virtual training world.

Origin:
ARC-7 created the base simulation as a sterile system.
ARC-7 calls it: THE SIM.

Then Mina and Kai get involved.
They mod it, style it, add stage vibes, add music, add drama.
They rename it proudly:
MEMEWARS ARENA.

Tone rule (locked):
No cute backgrounds.
No happy forests.
MemeWars arenas must feel intense, dark, and dangerous.

Why it matters:
MemeWars is designed so it can later become a real game world.
What you see in the show can become playable stages.


### Core Arena Rules


Rule 1 — Clear Fight Floor
Each arena must have a readable ground space:
- clear silhouettes
- strong contrast
- no clutter in the main fight zone

Rule 2 — Deep Background Atmosphere
Background should be dramatic:
- fog, rain, steam, sparks, distortion
- slow background movement
- bold lighting

Rule 3 — Strong Sound Identity
Each arena has a sound mood:
- low hums
- distant thunder
- metal creaks
- electric buzz
Sound must feel like a game.

Rule 4 — Stage Names
Every arena must have a name like a fighting game stage.
Fans should remember them easily.
