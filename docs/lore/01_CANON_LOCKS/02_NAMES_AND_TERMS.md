# Names and Terms (Canon Lock)

These words are locked and must stay consistent across all writing.

## Characters

| Name | Title | Description |
|------|-------|-------------|
| **Zaro** | Yellow Prime | Living spark of light condensed into a body |
| **ARC-7** | Cyan Prime | Watcher from the ARC Realm. Strict rules. Breaking them has a cost |
| **Mina** | Fireflies Operator | Teen strategist commanding drone swarm |
| **Kai** | MagDust Operator | Teen builder with electromagnetic glove |
| **The Black Hole** | K Prime | Black hole presence that consumes light |
| **Magenta Prime** | M Prime | Teased end of Book 1. Fully introduced Book 2. Female. Completes CMY |

## Key Terms

| Term | Definition |
|------|-----------|
| **Cosmic Fungus** | Official label on screens and ARC UI |
| **The Ink** | Casual human nickname for the Cosmic Fungus |
| **NOT OVER** | Zaro's defiance signal. Pure willpower, not magic |
| **Green** | C+Y combo. Appears only when Yellow and Cyan spiral fast enough |
| **Glow Burn** | Stamina crash when Zaro overuses his light |
| **Overclock** | ARC-7 pushes past safe limits. Correct spelling (not "Overlock") |
| **MemeWars** | Virtual training arenas built by ARC-7, enhanced by Mina and Kai |
| **Seed Core** | The source object Zaro protects and plants |
| **Spark Dust** | Residual glitter from the Container's impact |
| **Treasures Shelf** | Small objects Zaro has found and restored — personal and sentimental |

## Do Not Use

- "Void" — replaced by **Black Hole**
- "Mycelium" — too fancy
- "Overlock" — incorrect spelling (use **Overclock**)
- Complicated science words in dialogue

Keep dialogue simple. Let visuals carry meaning.
