# MEMEWARS RULES LOCK (CANON)


MemeWars is a virtual training world created by ARC-7.
ARC-7 builds the base simulation.
Mina and Kai later enhance it, style it, and name it “MemeWars.”

Tone rule (locked):
MemeWars backgrounds must NOT be cute or happy.
They must feel intense and serious, like a fighting game arena.
Dark mood, heavy sound, high focus.

Visual rule (locked):
Every MemeWars arena must feel like a game stage:
- clear fight floor
- deep background with atmosphere
- fog, rain, steam, sparks, or distortion
- strong lighting and shadows

Sound rule (locked):
MemeWars sound must feel like a game:
- clean hit sounds
- heavy thunks for blocks
- rising charge tones
- sharp release tones
Music is dark and driving, not cheerful.

System voice rule:
If we use a voice, it must be cold and minimal.
Examples:
READY. BEGIN. CLEAR.
No jokes. No cute lines.

Purpose rule (locked):
MemeWars is for training and skill growth.
It does not solve real-world threats.
It builds timing, teamwork, movement, stamina control.

Port-to-game rule (locked):
MemeWars arenas and move logic must be designed so they can be reused later
as real game stages and real game mechanics.
