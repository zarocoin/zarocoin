# ZaroVerse — Project Overview

## What ZaroVerse Is

ZaroVerse is a science-driven fantasy transmedia franchise built by Shihab Khalil. Core concept: "If light is hope — what happens when something in the universe wants to steal it?" Every power has a cost. Every win is earned. No infinite powers, no random miracles.

---

## Transmedia Lanes

The franchise expands across six lanes. Each has its own audience entry point. Lore powers all of them.

1. **Books** — Novel series. Book 1: *The Yellow Spark*, 10 chapters, published one at a time.
2. **Podcast** — Narrated chapters on Spotify.
3. **Animation** — PixAnime visual style (pixel-meets-anime).
4. **MemeWars** — Competitive game with 6 arenas (Dojo Void, Iron Furnace, Neon Rain Rooftop, Rift Hall, Silence Room, Black Rim Edge). Seeded by the Sim Arena in Book 1 Ch7.
5. **ShopZaro** — Merchandise.
6. **ZARO Coin** — ERC-20 token / community layer (separate voice lane — see `01_CANON_LOCKS/VOICE_SEPARATION_LOCK.md`).
