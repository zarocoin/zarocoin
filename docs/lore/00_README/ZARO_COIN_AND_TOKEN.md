## Hack and Buyback Story

The project was hacked. The attacker did not steal the coins outright — they sold them on Uniswap. Shihab bought them back from Uniswap into the company treasury over approximately 2 months. The recovered tokens were burned. Supply was then locked at 1 billion permanently.

This real-world story is baked into the lore:

| Real Event | Lore Parallel |
|---|---|
| Hack via trusted AI tool | Trojan amber crystal (Ch4-5) — the beautiful gift becomes the attack vector |
| Coins sold (not stolen) | Fruits SCORCHED on the tree (not scattered) — damaged in place |
| Most coins saved | Upper fruits survive, lower ones nearly die |
| 2-month buyback from Uniswap | Gradual tree recovery over weeks (top-down, like a chart recovering) |
| Bought back into treasury | Treasury = the tree. Fruits recover on the tree itself |
| Burned tokens / locked supply | Permanent tree scar. 1B fruits, no second generation |
| Founder resilience | NOT OVER — Zaro's defiance moment |
| Magenta's beam (Book 2 tease) | External help arriving — Y+M = Red energy seeding the lowest fruits |
