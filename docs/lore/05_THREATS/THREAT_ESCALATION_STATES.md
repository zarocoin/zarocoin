# THREAT ESCALATION STATES (SIMPLE)


This helps writers show the cosmic fungus getting worse in clear steps.

STATE 1 — TRACE
Small stains, tiny threads, quiet movement.
Easy to miss.
Screens show a few dots.

STATE 2 — OBSERVE
Fungus stays at a distance.
It watches patterns and learns behavior.

STATE 3 — SPREAD
Roots extend across surfaces.
More movement.
Screens show dots multiplying.

STATE 4 — TEST
Fungus tests barriers and edges.
It tries vents, cracks, corners, seams.
It looks for weak shields.

STATE 5 — TROJAN ENTRY
Fungus enters a protected space by hiding on an object.
Example (Season 1):
- a harmless-looking item is carried inside the home
- a small fungal trace is dormant outside
- it wakes up inside the warm safe space

Important:
The trojan object must feel believable. Not magic.
In Book 1: an amber crystal. Naturally beautiful.
Ink hid dormant spores inside (look like natural marble veins).
Zaro found it, thought it was natural, brought it inside himself.
Beauty was the trap. Good intentions were the vulnerability.

STATE 6 — DRAIN (THEFT)
Fungus touches a light source and steals it.
Glow dims.
This is tug-of-war, not punching.

Season 1 key target:
- stored fruits inside the house
- the main tree when Zaro glitches

STATE 7 — ARMORING / CONTROL
Fungus can infect a mechanical or ARC-type unit.
It weaves through the outer shell and creates an ARMOR LAYER.
The infected unit becomes AGGRESSIVE — the Ink hides inside,
using the unit as both shield and weapon.
Surface attacks hit the armor and bounce off.
Only focused, amplified energy can penetrate to the fungus within.

STATE 8 — CLEAR (TEMPORARY)
The heroes clear the main threat in a space.
The room feels safe again.

STATE 9 — TROJAN HITCHHIKE (LOCKED)
After the final battle (Chapter 10):
a microscopic Ink residue lodges in ARC-7's joints.
Too small for any scan to detect.
ARC-7 returns through the Veil carrying the hitchhiker.
Feels normal. Systems report clean. Reader sees what he cannot.

STATE 10 — INTERNAL BREACH
Fungus grows inside ARC systems.
Same alarms, but now inside.
This is the season-ending gut punch.
