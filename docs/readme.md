# ZARO Docs

This folder contains official documentation for the ZARO on-chain collectible project, including:

- Whitepaper with disclaimers
- Transparency summary
- Strategic roadmap
- Legal and regulatory statements

ZARO is a community-driven on-chain collectible tied to the ZaroVerse fictional universe, built on Ethereum, owned by no one, and operated through a decentralized community philosophy. It does not offer financial utility or investment potential — it is purely cultural.

See: [https://zaroverse.com](https://zaroverse.com)
